# PADWORLD ASSET GUIDE

The application is configured to automatically look for the files listed below in this `public/` folder.
If a file is found, it will be used. If not, the app will fall back to the default design.

## 1. HERO VISUAL
The Hero section looks for these files in priority order:
1. **Video**: `hero_bg.mp4`
2. **Image**: `hero_bg.jpg`

## 2. AUDIO
- **Background Music**: `padworld_theme.mp3`
- **Podcast/Briefing**: `padworld_podcast.mp3` (Upload your NotebookLM export here!)

## 3. SECTION DIVIDERS (IMAGES)
Upload images with these exact names to replace the dividers:

| Section | Filename | Description |
| :--- | :--- | :--- |
| **1. The Arena** | `section_1_court.jpg` | Smart Court, Stadium, Architecture |
| **2. Total Vision** | `section_2_vision.jpg` | Camera Lens, Optical Data, HUD |
| **3. Precision** | `section_3_referee.jpg` | Laser Lines, Geometry, Accuracy |
| **4. Neural Link** | `section_4_chat.jpg` | AI Brain, Network, Connection |
| **5. Deep Data** | `section_5_analytics.jpg` | Data Grid, Charts, Matrix |
| **6. Universe** | `section_6_ecosystem.jpg` | Global Map, Connectivity, Galaxy |
| **7. Future** | `section_7_outro.jpg` | Horizon, Sunrise, Space |

*Note: For best results, use JPG images with a resolution of at least 1920x1080.*
